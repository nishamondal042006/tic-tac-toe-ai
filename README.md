# Tic Tac Toe AI

A Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

- Two-player mode
- AI mode using Minimax Algorithm
- Win detection
- Draw detection
- Winning cell highlighting
- Reset Game and New Game buttons

## Technologies Used

- HTML
- CSS
- JavaScript

## What I Learned

- DOM Manipulation
- Event Handling
- Game State Management
- Recursion
- Minimax Algorithm
- Debugging JavaScript Applications
